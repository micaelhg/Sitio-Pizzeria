# Pizzeria

Aplicacion desarrollada con React y Vite como parte de los desafios del modulo de React en Desafio Latam.

El proyecto simula la interfaz de una pizzeria e incluye componentes reutilizables, formularios de autenticacion y renderizado dinamico de tarjetas de pizzas a partir de una base de datos local provisoria.

## Tecnologias utilizadas

- React
- Vite
- Bootstrap
- React Bootstrap
- JavaScript

## Funcionalidades actuales

- Barra de navegacion y pie de pagina.
- Vista principal con tarjetas de pizzas renderizadas dinamicamente.
- Formulario de registro con validaciones basicas.
- Formulario de inicio de sesion con validaciones basicas.
- Base de datos ficticia local en `src/utils/dbFicticia.js`.

## Estado actual

El proyecto sigue en construccion y avanza por hitos del curso. Actualmente, en `App.jsx`, la vista activa corresponde al formulario de login, mientras que otros componentes como `Home` y `Register` permanecen disponibles para seguir integrando nuevas etapas.
